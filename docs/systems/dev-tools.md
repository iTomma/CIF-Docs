# Dev Tools Integration

If the game's dev tools are enabled, the framework injects a **Custom Items** category into the spawn menu containing an entry for each registered custom item.

## Extra buttons

| Button | What it does |
|---|---|
| **Roll Armor** | Rolls a random armor drop for the current environment |
| **Roll Consumable** | Rolls a random consumable from the global loot table |
| **Roll Scavenge Loot** | Rolls from the scavenge table |

These buttons are useful for quickly testing loot weights and drop rates without having to kill enemies or open containers.
